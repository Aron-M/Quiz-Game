Version 1

Version 1 was very limited, with only 5 basic math questions answered in the same order giving the user a final score at the end. There was no other functionality at all, so there was a need to add new features and functionality.

  • There was no way for the user to exit the quiz without having to complete it first.
  • Only 5 questions, each the same and asked in the same order, making it very predictive.
  • The code logic was very basic and not very reusable. There were errors displaying the final score.
Version 2

Version 2 introduced new features, a cleaner layout and refactoring of the JavaScript code-logic in order to make it more dynamic and reuseable. I introduced Bootstrap styling for a cleaner look and to handle media queries as well as introduce interactive modals.

  • There needed to be a better way for user's to navigate through the application, whilst at the same time enjoy a bigger challenge with a bigger question-set being presented at random. Setting this up would allow for the addition of more question sets, perhaps even from different topics in the future. Keep an eye out for Version 3 in the near future!

New Features Added:
  • Users now have a way to opt out of the quiz to re-attempt or quit at any time, including a defensive modal to warn user when attempting to restart or quit.
  • Much wider selection of questions taken from a json file at random to avoid predictiveness.
  • Cleaner UX and front-end layout with the implimentation of Bootstrap styling.
  • The code-logic now refactored and more functionality introduced.
Version 3

Version 3 will come with some new key features where I will implement a real-time stats tracker box to the right of the questions panel. It will display a few key statistics that updates as the user progresses through the quiz. There will also be a 'High Socres' ranking list which will track the top scoring users' winning stats as show in the mockup images below. More questions will be added to choose 20 questions at random from a pool of 100!

New Features To Be Added:
  • Live score tracking and wrong answered questions displayed as the quiz progresses.
  • Current and final grade and percentage calculated and displayed.
  • Timer feature to display users' time taken to complete the quiz.
  • High Scores feature to display users' high scores and time taken to complete the quiz.
